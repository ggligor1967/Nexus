## Windows local E2E caveat

If `npx` or `npm run` fails with `ENOSPC` in `C:\npm-cache`, use a temporary npm cache under `C:\tmp` or clear/relocate the global npm cache.

```powershell
$env:npm_config_cache='C:\tmp\npm-cache-nexus'
```
