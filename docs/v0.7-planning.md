## Recommended Theme

```text
v0.7 - Artifact Quality + Revision UX
```

Nexus v0.6 proved that generated plans are validated, persisted, exportable, isolated, and CI-verified. The next increment should improve the usefulness and auditability of those artifacts without widening the product surface too aggressively.

```text
1. Better Markdown export template
2. Revision history UI
3. Plan comparison view
4. Selected prototype option marker
5. PDF export only if Markdown/revision flow remains stable
```

```text

- deployment automation
- enterprise CI/CD expansion

```

```text
Every v0.7 feature must preserve:

- typecheck/build pass
- E2E pass
- RLS isolation

- server-controlled artifact/write paths
- hosted GitHub Actions pass before promotion
```
